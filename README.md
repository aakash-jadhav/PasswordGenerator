# Password Generator

A React + Vite app for generating secure passwords.

## Live site

**[https://ciphercompiler.netlify.app/](https://ciphercompiler.netlify.app/)**

## Local development

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
npm run preview
```

## Stack

- [React](https://react.dev/)
- [Vite](https://vitejs.dev/)
